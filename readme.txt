This project was developed by Tejas Gupta, Shivam Gupta and Harshita Rastogi 
for DB Design Class at UTD in Fall 19

•	Developed a DBMS for a Cab Service which helps drivers and customers to book rides and assign rides to drivers
•	Designed EER Diagrams and Normalized the tables to avoid redundancy
•	Implemented stored procedures to calculate fare and ratings for rides and triggers to maintain integrity of database
Skills: SQL Developer, PL/SQL, ER Diagrams, Normalization
